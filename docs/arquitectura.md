# Arquitectura del Sistema

Documentación de la arquitectura del sistema SPM.

## Estructura del Monorepo

El proyecto está organizado como un monorepo con las siguientes áreas:

- **apps/**: Aplicaciones principales (frontend y backend)
- **packages/**: Paquetes compartidos (db, auth, utils)
- **docs/**: Documentación técnica

## Tecnologías

- **Frontend**: Next.js 14, React, TypeScript
- **Backend**: Node.js, Express, TypeScript
- **Base de Datos**: PostgreSQL (Neon)
- **ORM**: Prisma
- **Autenticación**: JWT
